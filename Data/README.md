## Data Description

This dataset contains the historical performance data used to build and evaluate the player forecasting model.

### Overview
- Scope: 2016–2025 MLB seasons
- Entries: 1,378 qualified hitter seasons
- Granularity: Player-season level

The dataset includes underlying performance metrics used for player evaluation and projection modeling. These metrics serve as the foundation for constructing player value scores, deriving aging curves, and identifying comparable players.

### Notes
Data is limited to qualified hitters based on minimum plate appearance thresholds
Players may appear multiple times across seasons to capture longitudinal performance trends
Outcome-based metrics (e.g., WAR, wRC+) are not used as primary inputs for valuation within the model
