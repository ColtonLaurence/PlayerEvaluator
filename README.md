## Player Evaluation & Forecasting Model
### Overview
This repository contains a customizable player evaluation and forecasting system designed to translate underlying performance metrics into long-term projections. The project integrates statistical analysis with on-field context to support data-informed decision-making in player development and roster construction.

A central principle of this work is that player value in baseball is inherently subjective. The model is therefore designed to be flexible, allowing users to define and weight the metrics that best align with their evaluation priorities.

### Repository Structure
- Model/
  Model implementation and forecasting pipeline
- Data/
  Dataset and supporting documentation

Additional details on methodology and data can be found in the respective README files within each directory.

### Model Overview
The model evaluates and projects players by combining:
- Customizable player valuation
- Recency-weighted performance
- Aging curve modeling
- Comparable player analysis
- Multi-year forecasting
This structure allows for a balance between flexibility, interpretability, and historical grounding.

### Data
The dataset spans 10 MLB seasons (2016–2025) and includes 1,378 qualified hitter seasons, structured at the player-season level to support longitudinal analysis.

### Purpose
This project reflects an interest in bridging statistical analysis with an applied understanding of the game. By integrating customizable valuation, historical trends, and comparable player outcomes, it provides a structured approach to player evaluation and development.

### Future Improvements

- Expand the model to include defensive and positional value
